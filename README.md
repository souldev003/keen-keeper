# 🍃 KeenKeeper

**KeenKeeper** is a relationship management tool designed to help you stay consistently connected with your loved ones, friends, and mentors. In the hustle of daily life, it's easy to lose track of time; this app ensures you never forget to reach out to the people who matter most.

---

## 📝 Description

KeenKeeper provides a personalized dashboard where you can set individual contact goals for each friend. The app tracks the number of days since your last contact and alerts you when it's time to reach out again, helping you maintain healthy and consistent social bonds.

---

## 🛠 Technologies Used

This project is built using a modern web stack:

- **Framework:** [Next.js 15](https://nextjs.org/) (App Router)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Icons:** [React Icons](https://react-icons.github.io/react-icons/)
- **Charts:** [Recharts](https://recharts.org/) (For interaction analytics)
- **Notifications:** [React-Toastify](https://fkhadra.github.io/react-toastify/)
- **State Management:** React Context API

---

## ✨ Key Features

### 1. Smart Relationship Tracking & Timeline

The app automatically calculates the days since your last interaction. Every contact (Call, Text, Video Call) is logged into a beautiful, filterable timeline, giving you a clear history of your social engagements.

### 2. Personalized Goals & Status System

Users can set specific contact frequencies for each friend (e.g., "Connect every 7 days"). The system categorizes relationships into statuses like **Overdue**, **On-track**, or **Needs Attention**, making it easy to prioritize who to call next.

### 3. Interactive Analytics Dashboard

Visualize your communication habits through an interactive Pie Chart that breaks down your interaction types. The "Quick Check-In" feature allows you to record new activities with a single click, instantly updating your stats and timeline.

## 🚀 Getting Started

1.  **Clone the repository:**

    ```bash
    git clone [https://github.com/souldev003/keen-keeper](https://github.com/souldev003/keen-keeper)
    ```

2.  **Navigate to the project directory:**

    ```bash
    cd keen-keeper
    ```

3.  **Install dependencies:**

    ```bash
    npm install
    ```

4.  **Run the development server:**
    ```bash
    npm run dev
    ```

Open [https://keen-keeper-inky.vercel.app/](https://keen-keeper-inky.vercel.app/) in your browser to see the application in action.
